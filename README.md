# KSPID - Kerbal Space Program Interactive Dashboard

KSPID (Kerbal Space Program Interactive Dashboard) is a Python-based tool that utilizes the kRPC library to provide a real-time interactive dashboard for monitoring and controlling various aspects of your Kerbal Space Program (KSP) missions.

## Features
- Real-time telemetry data from your spacecraft.
- Interactive dashboard displaying key metrics like altitude, speed, fuel levels, and more.
- Customizable alerts and notifications for mission-critical events.
- Simple control interface for executing basic commands.

## Requirements
- Kerbal Space Program (KSP) installed on your system.
- kRPC mod installed in KSP.
- Python 3.6 or higher.
- kRPC Python client library.
